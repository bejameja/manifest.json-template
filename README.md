# manifest.json-template
Use this template to create texture packs for Minecraft Bedrock Edition../
replace "UUID1" and "UUID2" wirh the [UUID generator](https://www.uuidgenerator.net/)



```json
{
  "format_version": 2,
  "header": {
    "name": "YOUR PACK NAME",
    "description": " ",
    "uuid": "UUID1",
    "version": [
      1,
      0,
      0
    ],
    "min_engine_version": [
      1,
      17,
      0
    ]
  },
  "modules": [
    {
      "type": "resources",
      "uuid": "UUID2",
      "version": [
        1,
        0,
        0
      ]
    }
  ]
}
